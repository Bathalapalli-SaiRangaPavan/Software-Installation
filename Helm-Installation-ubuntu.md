 ## Helm Installation
 ```
 curl -fsSL -o get_helm.sh https://raw.githubusercontent.com/helm/helm/master/scripts/get-helm-3
 ```
 ```
 chmod 700 get_helm.sh 
 ```
 ``` 
 ./get_helm.sh
 ```
## Helm Uninstallation 
 To see which folder its installed
 ```
 which helm 
 ```
 ```
 rm -rf /usr/local/bin/helm
 ```
